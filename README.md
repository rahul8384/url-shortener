# URL Shortener

A URL shortening service built with Java, Spring Boot, MySQL and Redis.

## Features
- Shorten long URLs using Base62 encoding
- Redirect to original URL via short code
- Track click count per URL
- Redis caching for fast redirects

## Tech Stack
- Java 17
- Spring Boot
- MySQL
- Redis

## Setup
1. Clone the repo
2. Create `application.properties` from `application.properties.example`
3. Set up MySQL and Redis
4. Run the application
